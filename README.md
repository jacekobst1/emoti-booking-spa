## Emoti Booking API

### How to start?

1. Run `bun install` (it can be also _pnpm_ of _npm_ - it's up to you)
2. Run `bun run dev`
3. Add new entry in _/etc/hosts_: `127.0.0.1 app.emoti-booking.test`
